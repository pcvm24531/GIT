#GIT DIFF
git diff => Compara el área de trabajo con el último commit. Muestra los cambios o diferencias que estan en la RAM(Stagin) y lo que esta en el disco.

git diff  commit1 commit2 => Muestra la diferencia entre 2 commits, los caracteres indican el numero del commit o identificados
                                                    Compara el commit más reciente con el commit que nosotros especifiquemos.

git diff rama1 rama2 => Comparar una rama con otra.

git diff commit1 commit2 nombreArchivo.ext => Muestra la diferencia del archivo entre ambos commits.

git diff HEAD => Compara el área de trabajo con una confirmación específica.

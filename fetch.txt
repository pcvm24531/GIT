#GIT FETCH, Permite traer cambios del repositorio remoto al repositorio local, pero no lo copia en nuestros archivos.
Seguido del fetch se debe hacer un git merge para actualizar nuestro repo local con los datos del repo remoto.
Para no usar ambos comando se puede usar git pull
git fetch --all => Obtien todas las ramas del repositorio remoto en el repositorio local, incluyendo tagas y commits
